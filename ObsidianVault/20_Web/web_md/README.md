# 20_Web - Web Markdown

Extracted web content as Markdown notes with Vision API OCR.

## Frontmatter Template
```yaml
---
source: web
captured_at: 2026-01-10T13:20:00+09:00
url: "https://example.com/page"
status: 200
content_hash: abc123
tags: [sitemap, web-crawl, ads]
---
```

## Content Sections
- Metadata (URL, status, canonical, H1)
- Extracted Text (OCR)
- Screenshot embed
- Links found
